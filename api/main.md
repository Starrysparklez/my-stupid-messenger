# General information

To access the API, you need to query the right version of the API.

| API version | Is active? | Deprecated? |
|-------------|------------|-------------|
| v1          | true       | false       |
|-------------|------------|-------------|

## API url

```
/api/<version>/...
```
for example
```
http://localhost:5000/api/v1/user/register
```
is used to register a new user using the 1st API version.
